# Legacy / NO USAR

Este directorio agrupa código histórico o de plantillas que ya no forma parte del MVP actual.
Se mantiene solo como referencia y no debe importarse ni modificarse al trabajar en las pantallas activas.

- `legacy-app/`: snapshot de la app antigua y artefactos de configuración.
- `root-navigation/`: navegadores y pantallas iniciales descartadas.
- `template-ui/`: componentes y hooks de plantilla Expo (incluye variantes `*-Amichos`).

Si necesitás recuperar algo de acá, copialo y ajústalo explícitamente fuera de `_legacy/`.
